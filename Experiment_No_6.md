# Experiment No 6

## Aim

List, Tuple, and Dictionary Operations

## Objective

Create and manipulate Python’s core data structures with insertion, deletion, and traversal.

## Process Flow

1. Create a list and perform append operation.
2. Create a tuple and print it.
3. Create a dictionary, add a key-value pair, and print it.

## Code

```python
# List
my_list = [1, 2, 3]
my_list.append(4)
print("List:", my_list)

# Tuple
my_tuple = (10, 20)
print("Tuple:", my_tuple)

# Dictionary
my_dict = {'name': 'Akshay', 'age': 32}
my_dict['city'] = 'Bhopal'
print("Dictionary:", my_dict)
```

---

## Output

```bash
List: [1, 2, 3, 4]
Tuple: (10, 20)
Dictionary: {'name': 'Akshay', 'age': 32, 'city': 'Bhopal'}
```
