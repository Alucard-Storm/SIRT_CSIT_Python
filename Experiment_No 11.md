# Experiment No 11

## Aim
Functions and Modular Programming

## Objective
Define and call functions with arguments and return values.

## Process Flow
1. Define a function `greet` taking a name argument.
2. Define a function `add` taking two arguments.
3. Call the functions and print the returned values.

## Code
```python
def greet(name):
    return f"Hello, {name}!"

def add(a, b):
    return a + b

print(greet("User"))
print("Sum of 5 and 3:", add(5, 3))
```
---

## Outcome

```bash
Hello, User!
Sum of 5 and 3: 8
```
