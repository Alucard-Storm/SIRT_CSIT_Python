# Experiment No 9

## Aim
Mean, Median, and Mode of a List

## Objective
Perform statistical computations using lists and conditional logic.

## Process Flow
1. Import the `statistics` module.
2. Define a list of numbers.
3. Calculate mean using `statistics.mean()`.
4. Calculate median using `statistics.median()`.
5. Calculate mode using `statistics.mode()`.
6. Print results.

## Code
```python
import statistics

data = [1, 2, 2, 3, 4, 7, 9]
print("Data:", data)
print("Mean:", statistics.mean(data))
print("Median:", statistics.median(data))
print("Mode:", statistics.mode(data))
```
---

## Outcome

```bash
Data: [1, 2, 2, 3, 4, 7, 9]
Mean: 4
Median: 3
Mode: 2
```
